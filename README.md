# mini-latent-diffusion
A custom PyTorch Latent Diffusion Model (LDM) built from scratch. Utilizes an SD-VAE for latent mapping, a CLIP text encoder for text conditioning, and a custom U-Net denoiser trained on the Flickr30k dataset via Classifier-Free Guidance. Includes full training and DDPM sampling pipelines. 🚀 Looking forward to your thoughts and feedback.😊
